Sass Basics

Variables

                    
                        $width: 5em;

                        .my-button {
                            $height: 2em;
                            width: $width;
                        }

                        .other-button {
                            height: $height; // won't work
                        }

Nesting

                    
                        .my-button {
                            background-color: papayawhip;

                            img {
                                outline: none;
                            }
                        }

Compiles to

                    
                        .my-button {
                            background-color: papayawhip;
                        }

                        .my-button img {
                            outline: none;
                        }

Parent selector

                    
                        .my-button {
                            background-color: papayawhip;

                            &:hover {
                                background-color: palevioletred;
                            }

                            img {
                                outline: none;
                            }
                        }

Compiles to

                    
                        .my-button {
                            background-color: papayawhip;
                        }

                        .my-button:hover {
                            background-color: palevioletred;
                        }

                        .my-button img {
                            outline: none;
                        }

Parent selector (2)

                    
                        .container {
                            .button {
                                .small & {
                                    font-size: 24px;
                                }
                            }
                        }

Compiles to

                    
                        .small .container .button {
                            font-size: 24px;
                        }

Mixins (Functions)

                    
                        @mixin border-radius($radius) {
                            -webkit-border-radius: $radius;
                               -moz-border-radius: $radius;
                                -ms-border-radius: $radius;
                                    border-radius: $radius;
                        }

                        .box {
                            @include border-radius(10px);
                        }

Compiles to

                    
                        .box {
                            -webkit-border-radius: 10px;
                            -moz-border-radius: 10px;
                            -ms-border-radius: 10px;
                            border-radius: 10px;
                        }

Important

Mixins return the block each time.
So every call will duplicate the whole code.

Extend/Inheritance

                    
                        %button {
                            font-family: Helvetica;
                            padding: 10px;
                        }

                        .button-primary {
                            @extend %button;
                            color: palevioletred;
                            background-color: papayawhip;
                        }

                        .button-secondary {
                            @extend %button;
                            color: papayawhip;
                            background-color: palevioletred;
                        }

Compiles to

                    
                        .button-secondary, .button-primary {
                            font-family: Helvetica;
                            padding: 10px;
                        }

                        .button-primary {
                            color: palevioletred;
                            background-color: papayawhip;
                        }

                        .button-secondary {
                            color: papayawhip;
                            background-color: palevioletred;
                        }

Calculations

                    
                        $grid-width: 960px;

                        .my-button {
                            width: 300px / $grid-width * 100%;
                        }

Compiles to

                    
                        .my-button {
                            width: 31.25%;
                        }

Important

These are compile time calculations.
No calculation on runtime values.

Imports

                    
                        // _reset.scss
                        html, body, ul, ol {
                            margin:  0;
                            padding: 0;
                        }

                        // main.scss
                        @import 'reset';

                        body {
                            font: 100% Helvetica, sans-serif;
                            background-color: #efefef;
                        }

Compiles to

                    
                        html, body, ul, ol {
                            margin:  0;
                            padding: 0;
                        }

                        body {
                            font: 100% Helvetica, sans-serif;
                            background-color: #efefef;
                        }

Important

  • @import will just dump the content of the file
  • No dependency graph -> duplication of every imported file
  • Files which emit styles should never be imported more than once
  • There are packages which fix this behaviour:
    https://github.com/wilsonpage/sass-import-once
    https://www.npmjs.com/package/node-sass-import-once
