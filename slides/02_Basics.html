Box Model

padding, border, margin

box-sizing

What gets included in width and height calculation

  • content-box
  • border-box

content-box

  • only content of the element is included
  • padding, border and margin are added on top

border-box

  • content, padding and border are included
  • only margin is added on top
  • mostly the setting we want but not default
                    
                        * {
                            box-sizing: border-box;
                        }
